## ✨ Les 3 Points Clés

### 1️⃣ CSS Prime Sur Background JavaFX
```
En JavaFX, le style CSS a une PLUS HAUTE priorité 
que le Background défini en Java.
```

### 2️⃣ Comment Fixer
```java
pane.setStyle("");              // Retirer le CSS
pane.setBackground(image);      // Appliquer l'image
```

### 3️⃣ Prêt à Utiliser
```bash
mvn javafx:run
```
