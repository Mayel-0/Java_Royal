# Java Royal

Projet JavaFX regroupant plusieurs mini-jeux (Sudoku, Pendu, Flappy Bird, etc.).

## Prerequis
- Java 21+
- Maven (ou le wrapper `./mvnw`)

## Lancer l'application
```zsh
cd /Users/mayel/Documents/Ynov/Java/Java_Royal
./mvnw javafx:run
```

## Tests
```zsh
cd /Users/mayel/Documents/Ynov/Java/Java_Royal
./mvnw test
```
