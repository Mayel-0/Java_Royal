# 🚀 Quick Start - Leaderboard

## ⚡ En 5 minutes

### 1️⃣ Vérifier l'installation
```bash
cd /Users/mayel/Documents/Ynov/Java/Java_Royal
bash leaderboard_setup.sh
```

### 2️⃣ Compiler
```bash
mvn clean compile
mvn clean package
```

### 3️⃣ Lancer l'app
```bash
mvn javafx:run
```

### 4️⃣ Tester
1. Se connecter
2. Cliquer "Classement" (bouton doré)
3. Voir les meilleurs joueurs ✅
